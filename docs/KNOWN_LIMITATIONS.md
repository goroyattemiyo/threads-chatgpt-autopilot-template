# v0.1 既知の制限

- 対応SNSはThreadsのみです。
- 投稿時刻はGitHub Actionsの開始遅延により、設定時刻からずれる場合があります。
- 投稿時刻は秒単位・分単位で保証しません。
- 1つの投稿枠に複数の投稿を入れた場合、1回の確認で処理されるのは原則1件です。
- ツリーの途中で失敗した場合、自動で安全に途中から再開する機能はまだありません。
- 親投稿だけ成功して返信に失敗した場合は、手動確認が必要です。
- 投稿削除の自動化は含まれていません。
- インサイトの自動取得は含まれていません。
- OpenAI APIによる投稿文・画像の自動生成は含まれていません。
- 画像付き投稿では、投稿用画像をPublicリポジトリへ置く標準構成です。
- Public画像リポジトリへ置いた画像は、投稿前でもインターネット上から閲覧可能です。
- 外部サービスの障害・仕様変更中は正常動作しない場合があります。
- Metaによるアカウント制限や投稿審査を解除する機能はありません。
- 顧客環境への導入リハーサルと長期連続運転テストは未完了です。
